# Инвентаризация сайта электромонтаж.kz

## URL и маршруты

| Маршрут | Описание |
|---------|----------|
| `/` | Главная (все секции на одной странице) |
| `/services` | Услуги (дублирует секцию с главной, отдельная страница для навигации) |
| `/prices` | Прайс-лист / условия работы |
| `/contacts` | Контакты и форма заявки |
| `/calculator` | Калькулятор стоимости (standalone или секция) |

Внешние ссылки с текущего сайта:
- Прайс: `/pricelist.html` → в SPA: `/prices`
- Калькулятор: `/calculator-standalone.html` → в SPA: `/calculator`

## Формы

### 1. Форма заявки (контакты / футер)

| Поле | Тип | Обязательное | Валидация |
|------|-----|--------------|-----------|
| Имя | text | да | required |
| Телефон | tel | да | required, phone |
| Email | email | нет | email |
| Тип проекта | select | нет | Квартира, Частный дом, Офис, Коммерческое помещение, Другое |
| Сообщение | textarea | да | required |

Кнопка: «Отправить заявку». Поведение: отправка через mock → toast/modal «Успешно отправлено».

### 2. Форма «Точный расчёт» (из калькулятора)
- Те же поля или подмножество (имя, телефон, email, тип объекта, площадь/комментарий).
- Отправка через тот же `submitLead` mock.

### 3. ИИ-ассистент
- Поле ввода сообщения + «Прикрепить файлы» (визуально, без реальной загрузки).
- «Оформить заявку» ведёт на форму заявки или модалку с формой.

## Виджеты

| Виджет | Расположение | Поведение |
|--------|--------------|-----------|
| **Калькулятор** | Страница `/calculator`, на главной — CTA-блок | Поля (площадь, тип объекта и т.д.) → расчёт по правилам из calculatorConfig. Кнопка «Точный расчёт» открывает форму. |
| **ИИ-ассистент** | Плавающая кнопка справа снизу + секция на главной | Открывает чат. Ответы по ключевым словам (mock). Fallback → форма заявки. |
| **Модалки** | По триггерам | Успешная отправка формы; при необходимости «Оформить заявку» из ассистента. |
| **Toast** | Глобально | «Успешно отправлено» после submit формы. |

## Базовые UI-компоненты

- Header (лого, навигация, контакты/кнопка)
- Footer (контакты, соцсети, копирайт)
- Button (primary, secondary, outline)
- Input, Textarea, Select
- Card (услуги, преимущества, этапы)
- Modal
- Toast / Notification
- Accordion (FAQ при наличии)
- MainContainer (обёртка контента с отступами и max-width)

Документ позволяет воссоздать структуру и контент сайта без доступа к исходному коду.
